Cloudbox: Options in accounts.yml



Parameter Sub-Parameter Definition Required Choices Default Notes
user User account for the server. yes seed If the user account does not already exist, it will be created during install.
passwd Password for the user account. yes Note 1: Password must be in alphanumeric characters. No special characters are allowed.

Note 2: This password is also used for NZBGet, ruTorrent, and NZBHydra2.

email E-mail address. yes This will be used to 1) Register the Let's Encrypt SSL certificates, and 2) Receive certificate expiration notices.

This will also be used for Cloudflare authentication (i.e. this must match the e-mail address used in the Cloudflare account profile).
cloudflare_api_token Cloudflare API Token. no Fill this in to have Cloudbox add subdomains on Cloudflare, automatically; leave it blank, to have all Cloudflare related functions disabled.

Note: CDN (i.e. proxy) will not be turned on, by default, but you may turn them on later (see here).
plex Plex Credentials. no This will be used to 1) claim the Plex server under your username, and 2) generate Plex Access tokens for apps such as Plex Autoscan, etc.
user Plex Username or email address.
pass Plex Password.
pushover Pushover settings. no This will be used to send out messages during certain tasks (e.g. backup).

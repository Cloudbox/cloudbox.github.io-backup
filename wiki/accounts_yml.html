Cloudbox: Options in accounts.yml



Parameter Sub-Parameter Definition Required Choices Default Notes
user User information. yes If the user account does not already exist, it will be created during install.
name User name for the server. seed If the user account does not already exist, it will be created during install.

Also used to create first-time logins for NZBGet, ruTorrent, NZBHydra2, and potentially other apps.
pass Password for the user account. Sets password for the server's user account when creating a new account. This will not change the password of an existing one.

Also used to create first-time logins for NZBGet, ruTorrent, NZBHydra2, and potentially other apps.

email E-mail address. This will be used to 1) Register the Let's Encrypt SSL certificates, and 2) Receive certificate expiration notices.

This will also be used for Cloudflare authentication (i.e. this must match the e-mail address used in the Cloudflare account profile).
cloudflare Cloudflare account. no Fill this in to have Cloudbox add subdomains on Cloudflare, automatically; leave it blank, to have all Cloudflare related functions disabled.

Note: CDN (i.e. proxy) will not be turned on, by default, but you may turn them on later (see here).
email E-mail address used for the Cloudflare account.
api API Token.

pass Plex Password.
